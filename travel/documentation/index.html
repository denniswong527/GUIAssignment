Installation

Follow the following steps to get started with your Travel Gateway Template:

  1. Extract the zip Folder to find all the Files
  2. Uplaod these files to your liver server by using FTP or cPanel
  3. Make sure you upload the following files or folders on your live server
    • travel/assets - contains all the assets such as js, images, css
    • travel/html - contain all html pages
    • travel/index.html - homepage

The other files or plugins can be used according your requirements and you can find all the 3rd party plugin in assets/lib folder.
Now all done you can start creating beautiful pages and widgets

File Structure

We follow a simple and easy to use file structure.

                        
Travel Gateway

    -assets  contain all the css, img and js files
            --css     core css files
            --fonts  core template fonts files
            --img  core template fonts files
            --js      core javascript files

    -html  all html pages
        --fight     all flights page
        --home  All index and home pages
        --hotel all hotels page
        --pages     different home styles
        --shortcodes template shortcodes
        --travel     all travels page

    -index.html  template homepage

Customizations

You can create a new custom.css file in css folder so that you can write custom styles in that file to implement customization. Simply add all your css files before the custom.css file and link it at the end of document . Make sure that this is the last linked CSS File in the Document so that your other css styles are Overwritten properly.

                        
<link rel="stylesheet" href="css/style.css" type="text/css" />
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1" />


Here goes your custom.css
=============================================
<link rel="stylesheet" href="css/custom.css" type="text/css" />

SASS Setup

There is a sass folder in assets/css. You can find all the sass files in that folder there are three sass files in sass folder.

  1. style.scss main sass file
  2. screen.scss contain media quries for changing styles in different screens
  3. variable.scss contain template variables for fonts and colors

you can use cmd to compile file don't need to watch variables.scss and screen.scss files because these two files are imported in main app.scss file you don't need to include these two files into your project. Only link app.css file to your project

Search style One

You can choose between 3 different types of headers while creating your page. Simply use the relevant search ID to the header tag to activate any search. here is the sample code for search type one:

                        
<div class="main-fill">

    <div class="panel">

      <div class="panel-heading">

        <ul class="nav nav-tabs">

          <li class="active"><a href="#tabtravel" data-toggle="tab"> Travel </a></li>

          <li class="active"><a href="#tabtravel" data-toggle="tab"> Flight </a></li>

          <li class="active"><a href="#tabtravel" data-toggle="tab"> Hotels </a></li>

        </ul>

      </div>

    </div>

</div>
